Tiles Gallery
A modern e-commerce platform for premium tiles, built with Next.js and BetterAuth. Browse products, view detailed specifications, manage your cart, and securely manage user accounts.

Features :

-Responsive product catalog with category filtering
-Detailed tile product pages with specifications and reviews
-Shopping cart with quantity management
-Secure authentication via BetterAuth
-User profile management and session handling
-Sticky navigation with active route highlighting
-Optimized for performance and SEO

Tech Stack :

Framework: Next.js 16 (App Router)
Language: TypeScript
Authentication: BetterAuth
Database: MongoDB Atlas
Styling: CSS-in-JS
Deployment: Netlify

Getting Started

Prerequisites :

Node.js 18+
MongoDB Atlas account or local MongoDB instance
npm or yarn
